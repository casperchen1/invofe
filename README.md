A terminal based file explorer.

![demo](./resources/assets/demo.webp)
## Features
- File navigation
- Display files
- and more to come!
## Build Project
1. Create a build folder at the project root directory
`mkdir build`
`cd build`
2. Generate CmakeFiles
`cmake ..`
3. Build
`cmake --build .`
4. Execute
`.\Debug\InvoFE.exe`
